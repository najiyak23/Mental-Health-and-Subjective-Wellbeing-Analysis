# Mental Health and Subjective Wellbeing Analysis

This project presents a statistical analysis of a population survey from Cali, Colombia, examining the relationship between mental health symptoms and subjective wellbeing. The analysis is exploratory and observational, with the goal of identifying stable patterns rather than estimating causal effects.

## Research Question

How are depression, anxiety, and stress associated with subjective wellbeing in the survey data from Cali, Colombia? This project also examines whether these relationships remain stable across different statistical checks and model specifications.

## Dataset

The dataset used in this project is: [mentalhealth_swb_2425v5sheet1.csv](https://bsos-data.umd.edu/dataset/c2986256-9120-4e59-a72b-53b708774e83/resource/bdc908d9-0175-48e6-b015-a34f602ffce0/download/mentalhealth_swb_2425v5sheet1.csv)

## Methods

- Descriptive statistics
- Correlation analysis
- Regression models
- Prediction checks
- Robustness tests, including outlier analysis and quantile regression

## Main Findings

The results show a consistent negative association between mental health symptoms and subjective wellbeing. Depression, anxiety, and stress are all linked to lower wellbeing, while demographic variables explain comparatively less variation. The findings are stable across multiple checks, which increases confidence in the overall pattern.

## Limitations

This study is based on cross-sectional survey data, so it cannot identify causal effects. The data are self-reported, which may introduce measurement error or reporting bias. The results should therefore be interpreted as associations rather than evidence of intervention impact.
